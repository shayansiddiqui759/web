Enhancements

Embedded Map Feature on Index.html

Added an interactive Google Maps feature to the Contact section of the index.html page. This allows users to locate the business address easily.

Source: Google Maps Embed API and HTML

Horizontal Scrolling for Service Card Section

Implemented a horizontal scrolling feature for the Service Section on the index.html page. This enhancement provides an elegant way to showcase multiple services when there are more than three service cards.

The service cards are displayed within a horizontal scrolling container, allowing users to effortlessly navigate through the services using a horizontal scroll bar.

Source: Custom CSS styles added to the style.css file.

How to Use:

  1. Visit the index.html page on the website.
  2. Navigate to the "Our Services" section.
  3. If there are more than three service cards, you'll see a horizontal scroll bar below the service cards.
  4. Scroll horizontally to explore additional service cards beyond the initial view.
