Enhancements

Implemented Slide Show

Implemented slideshow feature on the index.html page. This enhancement provides an elegant way to showcase multiple services when there are more than three service cards.

Link to the video on youtube

Source: Thapa Technical

Use of Query Selector

Use `querySelectorAll()` on the jobs.html and apply.html to make the website responsive to user actions.

Source: Custom CSS styles added to the style.css file.

How to Use:

  1. Visit the index.html page on the website.
  2. Navigate to the "Our Services" section.
  3. If there are more than three service cards, you'll see a horizontal scroll bar below the service cards.
  4. Scroll horizontally to explore additional service cards beyond the initial view.
